# MixCanvas Studio

Konva 기반 인터랙티브 캔버스 플레이그라운드입니다. 이미지, 유튜브 영상, 프리셋 도형을 드래그앤드롭으로 올려두고 빠르게 배치 실험을 해볼 수 있습니다.

## 시작하기

```bash
npm install
npm run dev
```

`http://localhost:3000` 에 접속해 변경 내용을 바로 확인하세요.

## 주요 기능

- 이미지·유튜브 영상 드롭 및 배치
- 프리셋 도형(파란/초록 사각형, 빨간 원, 제목 텍스트) 추가
- Konva 기반의 깔끔한 렌더링과 인터랙션
